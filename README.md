​	这是一个由springboot+vue实现的简易的博客，项目上采用前后端分离架构。展示上分为前台与后台，前台用于展示博文，后台用于管理博文/分类/个人介绍/实验室等。

适用于：

* 学习前后端分离开发

* 二次开发
    
    相关组件版本: mysql5.7.32   |   node12.15.0  
    
    项目开发总结参见：summary.md
