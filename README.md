这是一个由springboot+vue实现的简易的博客，项目上采用前后端分离架构。

展示上分为前台与后台，前台用于展示博文，后台用于管理博文/分类/个人介绍/实验室等。

本项目是博客的后端部分，前端部分[参见](https://github.com/DianeDii/blog_vue)。

---

除基本功能外还支持：

* 全局搜索
* 文章加密（AES）
* 待更新...
